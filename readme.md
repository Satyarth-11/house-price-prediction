# House Price Prediction 🏠

This project is a Machine Learning web application that predicts house prices based on user inputs such as location, BHK, number of bathrooms, and total square feet.

## Tech Stack

* Python
* Flask
* Pandas
* NumPy
* Scikit-Learn
* HTML/CSS

## Features

* Predict house price based on property details
* Simple web interface using Flask
* Trained machine learning model

## How to Run

1. Clone the repository

```
git clone git clone https://github.com/Satyarth-11/house-price-prediction.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```

4. Open in browser

```
http://127.0.0.1:5001/
```
